# Job Portal Application

A full-stack Job Search Web Application inspired by Indeed and Naukri.

## Features
- User Signup & Login
- Job Search (real-time filtering)
- Add Job
- Apply for Jobs
- Track Applied Jobs

## Tech Stack
- Frontend: React (TypeScript)
- Backend: Node.js, Express
- Database: MongoDB

## Run Locally

### Backend
cd backend
npm install
npm start

### Frontend
cd frontend
npm install
npm start
